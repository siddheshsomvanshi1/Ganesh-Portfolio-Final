# Modern DevOps Portfolio

A professional, high-performance portfolio website built with React and Vite, specifically designed for DevOps Engineers and Cloud Professionals.

## 🚀 Features

- **Dynamic Theming**: Three distinct professional themes (Midnight Slate, Cyber Dusk, and Arctic Snow) with persistent storage.
- **Advanced Animations**: Smooth scroll reveals, staggered grid entrances, 3D hover effects, and a custom scroll progress bar.
- **Responsive Design**: Fully optimized for all devices, from large desktop monitors to mobile phones.
- **Glassmorphism UI**: Modern aesthetic using backdrop blurs and subtle gradients.
- **Interactive Components**: Scroll-spy navigation, back-to-top functionality, and a technical terminal visualizer.

## 🛠️ Tech Stack

- **Framework**: [React.js](https://reactjs.org/) (v18+)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: Modern CSS3 with Variables & Keyframes
- **Icons**: [FontAwesome](https://fontawesome.com/)
- **Fonts**: Space Grotesk & Inter

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (Latest LTS recommended)
- npm (comes with Node.js)

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   ```

2. Navigate to the project folder:
   ```bash
   cd portfolio-react
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the local development server:
```bash
npm run dev
```
The application will be available at `http://localhost:5173`.

### Production Build

Create an optimized production build:
```bash
npm run build
```
The output will be generated in the `dist/` folder.

## 🎨 Customization

### Changing Themes
Themes are managed via CSS variables in `src/index.css`. You can modify colors for `midnight`, `cyber`, and `arctic` themes under the `:root` and `[data-theme]` selectors.

### Updating Projects & Skills
Edit the content arrays within `src/App.jsx` to update your personal information, technical skills, projects, and certifications.
